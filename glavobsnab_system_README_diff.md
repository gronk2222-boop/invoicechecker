# 🏗️ Система интеллектуального сравнения счетов для "Главоблснаб"

Веб-приложение для автоматического сравнения входящих счетов от поставщиков с исходной заявкой клиента.

## 🚀 Возможности

- **Парсинг документов**: PDF, XLS/XLSX, DOC/DOCX, CSV, TXT
- **Сравнение позиций**: Точное совпадение, аналоги, отсутствующие и лишние позиции
- **ИИ-поиск характеристик**: Автоматическое определение веса, объема, стандартов
- **Проверка совместимости**: Предупреждения о несовместимых материалах
- **Расчет логистики**: Подбор транспорта, расчет стоимости доставки
- **История цен**: Анализ динамики цен по поставщикам
- **Визуализация**: Карта точек загрузки и выгрузки
- **Экспорт**: CSV и Excel отчеты

## 📋 Требования

- Python 3.10+
- Streamlit
- pandas, openpyxl, pdfplumber, python-docx
- folium, streamlit-folium

## 🔧 Установка

```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск приложения
streamlit run app.py
```

## 🎯 Пользовательский сценарий

### Шаг 1: Заявка клиента
1. Загрузите файл заявки (любой поддерживаемый формат)
2. Укажите имя клиента
3. Укажите адрес доставки

### Шаг 2: Счета от поставщиков
1. Загрузите один или несколько счетов от разных поставщиков
2. Система автоматически распознает поставщика и номенклатуру

### Шаг 3: Результаты
1. Сравнение позиций с цветовой индикацией статусов
2. Расчет логистики и стоимости доставки
3. Проверка совместимости товаров
4. Анализ истории цен
5. Карта с точками поставщиков
6. Экспорт результатов

## 🎨 Интерфейс

- **Темная тема**: Фон `#0b132b`, акценты `#87ceeb`
- **Пошаговый мастер**: Прогресс-бар обработки
- **Интерактивная карта**: Folium с маркерами поставщиков
- **Статусы позиций**:
  - ✅ Точное совпадение
  - ⚠️ Разница в количестве/цене
  - ❌ Отсутствует в счете
  - 🆕 Лишняя позиция

## 📊 База данных

SQLite база данных хранит:
- Клиентов и поставщиков
- Документы (заявки и счета)
- Позиции номенклатуры
- Историю цен
- Логи обработки

Файл БД: `glavobsnab.db` (создается автоматически)

## 🔍 Парсинг документов

Система автоматически извлекает:
- Наименование товара
- Количество и единицы измерения
- Цену за единицу и общую сумму
- Артикулы и ГОСТы
- Информацию о поставщике
- Условия доставки

## 🚚 Логистика

Автоматический подбор транспорта:
- Газель (1.5т / 9м³)
- Газель удлиненная (1.5т / 16м³)
- 5-тонник (5т / 36м³)
- 10-тонник (10т / 56м³)
- 20-тонник (20т / 86м³)
- Манипулятор 10т (10т / 40м³)

## ⚠️ Проверка совместимости

Система предупреждает о:
- Отсутствии сопутствующих товаров
- Потенциальных конфликтах материалов
- Несовместимых комбинациях

## 📈 История цен

При каждой обработке:
- Сохранение цен в историю
- Расчет средней цены
- Определение тренда (рост/падение/стабильно)
- Сравнение с предыдущими закупками

## 💾 Экспорт

- **CSV**: Сводка по поставщикам с итогами
- **Excel**: Детальное сравнение всех позиций
